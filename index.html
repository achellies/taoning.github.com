iOS

iOS刚刚起步

  1. Unrecognized selector

    GitHub

    在调用performSelector时如果忘记了检查selector是否存在(respondsToSelector)，则很容易造成unrecognized selector exception, 这里提供一个简单方便的方案解决unrecognized selector报错的问题。

  2. Advance Objective-C

    GitHub

    探讨Objective－C的高级语法，例如：Swizzing、shallow copy/deap copy、dynamic method resolution、message forwarding、keep unrecognized selector crash、dynamic create protocol & class、多播代理、PromiseKit的使用等。

  3. GestureZoomAndPanAndRotate

    GitHub

    UIPinchGestureRecognizer UIPanGestureRecognizer UIRotationGestureRecognizer for any UIView。
